Create `internal/browser/pages/login.go`:
- Package `pages`
- Define `LoginPage` struct with `page playwright.Page` field.
- `func NewLoginPage(page playwright.Page) *LoginPage`
- `func (lp *LoginPage) Login(username, password string) error`:
  1. Navigate: `lp.page.Goto("https://dns.he.net/")`.
  2. Fill email: `lp.page.Locator(SelectorLoginEmail).Fill(username)`.
  3. Fill password: `lp.page.Locator(SelectorLoginPassword).Fill(password)`.
  4. Click submit: `lp.page.Locator(SelectorLoginSubmit).Click()`.
  5. Wait for load: `lp.page.WaitForLoadState(playwright.LoadStateNetworkidle)`.
  6. Verify success: check `lp.page.Locator(SelectorLogoutLink).IsVisible()`. If not visible, return error "login failed: logout link not visible after login" (do NOT include username/password in error).
  7. Return nil on success.
- `func (lp *LoginPage) IsLoggedIn() (bool, error)`:
  1. Try navigating to `https://dns.he.net/` (in case page is on an error or unrelated page).
  2. Wait for load state.
  3. Return `lp.page.Locator(SelectorLogoutLink).IsVisible()`.

  1. `zl.page.Goto("https://dns.he.net/")`.
  2. Wait for load state.
  3. Wait for domains table: `zl.page.Locator(SelectorDomainsTable).WaitFor()`.
- `func (zl *ZoneListPage) ListZones() ([]model.Zone, error)`:
  1. Get all delete images (they have zone name and ID): `zl.page.Locator(SelectorZoneDeleteImg).All()`.
  2. For each delete img: extract `name` attribute (zone name) and `value` attribute (zone ID).
  3. Build and return `[]model.Zone{ID: value, Name: name}`.
- `func (zl *ZoneListPage) GetZoneID(zoneName string) (string, error)`:
  1. Build selector: `fmt.Sprintf("img[alt=\"delete\"][name=\"%s\"]", zoneName)`.
  2. Get `value` attribute. Return the zone ID string.
- `func (zl *ZoneListPage) NavigateToZone(zoneID string) error`:
  1. Navigate to: `fmt.Sprintf("https://dns.he.net/?hosted_dns_zoneid=%s&menu=edit_zone&hosted_dns_editzone", zoneID)`.
  2. Wait for load state.
  3. Verify: wait for at least one `tr.dns_tr` or `tr.dns_tr_locked` (SOA is always present in a zone).
- `func (zl *ZoneListPage) GetRecordRows() ([]RecordRow, error)`:
  1. Define `RecordRow` struct: ID string, DisplayText string, IsLocked bool.
  2. Get all `tr.dns_tr` rows.
  3. For each row: extract `id` attribute (record ID), get inner text of the display cell.
  4. Also get `tr.dns_tr_locked` rows as locked records.
  5. Return combined slice.

Create `internal/browser/pages/recordform.go`:
- Package `pages`
- Import `internal/model` for Record type.
- Define `RecordFormPage` struct with `page playwright.Page` field.
- `func NewRecordFormPage(page playwright.Page) *RecordFormPage`

Field mapping by record type (from CONTEXT.md):
- **Standard (Name + Content + TTL):** CNAME, ALIAS, NS, CAA, HINFO, RP, LOC, NAPTR, PTR, SSHFP, SPF
- **With DDNS checkbox (Name + Content + TTL + dynamic):** A, AAAA, TXT, AFSDB
- **MX (Name + Priority + Content + TTL):** MX
- **SRV (Name + Priority + Weight + Port + Target + TTL):** SRV -- NOTE: SRV has NO Content field, uses Target instead

- `func (rf *RecordFormPage) OpenNewRecordForm(recordType string) error`:
  1. Execute JS to trigger the form: `rf.page.Evaluate("editFormHandler", recordType)`.
  2. Wait for the edit form to be visible: `rf.page.Locator(SelectorRecordForm).WaitFor()`.

- `func (rf *RecordFormPage) FillRecord(rec model.Record) error`:
  1. Open form for record type: `rf.OpenNewRecordForm(rec.Type)`.
  2. Fill Name field: `rf.page.Locator(SelectorRecordName).Fill(rec.Name)`.
  3. Based on rec.Type, fill the appropriate fields:
     - If SRV: fill Priority, Weight, Port, Target (NOT Content).
     - If MX: fill Priority, Content.
     - If A, AAAA, TXT, AFSDB: fill Content, optionally check Dynamic checkbox.
     - All others: fill Content.
  4. Select TTL: `rf.page.Locator(SelectorRecordTTL).SelectOption(playwright.SelectOptionValues{Values: playwright.StringSlice(strconv.Itoa(rec.TTL))})`. NOTE: TTL select uses string values like "300", "3600", etc.
  5. Return nil.

- `func (rf *RecordFormPage) SubmitRecord() error`:
  1. Click submit: `rf.page.Locator(SelectorRecordSubmit).Click()`.
  2. Wait for load: `rf.page.WaitForLoadState(playwright.LoadStateNetworkidle)`.
  3. Return nil.

- `func (rf *RecordFormPage) FillAndSubmit(rec model.Record) error`:
  Convenience method: FillRecord + SubmitRecord.

- `func (rf *RecordFormPage) EditExistingRecord(recordID string) error`:
  1. Click the record row to open edit form: `rf.page.Locator(fmt.Sprintf("tr#%s", recordID)).Click()`.
  2. Wait for edit form visible.

- `func (rf *RecordFormPage) DeleteRecord(recordID, zoneName, recordType string) error`:
  1. Find the delete cell in the record row.
  2. Execute the deleteRecord JS function: `rf.page.Evaluate("deleteRecord", recordID, zoneName, recordType)`.
  3. Wait for page reload/confirmation.

- `func (rf *RecordFormPage) fieldMappingForType(recordType string) []string`:
  Return which selector fields are relevant for each record type. This is a helper for validation and for knowing which fields to fill. Use a switch statement matching all 17 types.

Update `internal/browser/session.go` to replace the ensureHealthy stub (from Plan 01-02) with real health check and recovery logic.

Update `ensureHealthy(ctx context.Context, session *AccountSession) error`:
1. **No context exists (first use):** If session.ctx is nil, create a new BrowserContext via `sm.launcher.NewAccountContext(opTimeoutMs)`, create a new page, perform login (fetch credentials from sm.credProvider, call pages.NewLoginPage(page).Login(cred.Username, cred.Password)), set session fields (ctx, page, lastLogin=time.Now(), healthy=true). Log "new session created" with accountID.

2. **Session too old (proactive re-login):** If `time.Since(session.lastLogin) > sm.reloginAge`, close the old BrowserContext, create fresh context+page, re-login. Log "session aged out, re-logging in" with accountID and age. This prevents silent session expiry.

3. **Health check (session exists but might be stale):** Use `pages.NewLoginPage(session.page).IsLoggedIn()` to check if the logout link is visible. If check fails OR returns false:
   a. Log "session unhealthy, recovering" with accountID and error.
   b. Close old BrowserContext.
   c. Create fresh context+page.
   d. Login again.
   e. Update session fields.
   f. If login fails, return error wrapping ErrSessionUnhealthy.

4. **Session healthy:** Return nil.

CRITICAL wiring: Import `internal/browser/pages` in session.go. The pages package uses selectors from selectors.go to interact with dns.he.net. The session.go uses pages.LoginPage for health checks. This creates the flow: session.go -> pages/login.go -> selectors.go -> dns.he.net.

CRITICAL: When fetching credentials in ensureHealthy, NEVER log the password. Log only accountID and username. Use slog with explicit field names to avoid accidental credential leakage.

- Public method to force session refresh for a specific account.
- Acquires the per-account mutex (via WithAccount pattern).
- Closes existing context, creates new one, logs in fresh.
