**AddZone(domainName string) (string, error)**

Steps:
1. Navigate to zone list: call `zp.NavigateToZoneList()`. Return error if it fails.
2. Click the trigger: `zp.page.Locator(SelectorAddZoneTrigger).Click()`. Return wrapped error on failure.
3. Wait for panel: `zp.page.Locator(SelectorAddZonePanel).WaitFor()`. Return wrapped error on failure.
4. Fill domain name: `zp.page.Locator(SelectorAddZoneInput).Fill(domainName)`. Return wrapped error.
5. Click submit: `zp.page.Locator(SelectorAddZoneSubmit).Click()`. Return wrapped error.
6. Wait for networkidle: `zp.page.WaitForLoadState(playwright.PageWaitForLoadStateOptions{State: playwright.LoadStateNetworkidle})`.
7. Look up the new zone ID: call `zp.GetZoneID(domainName)` — this reuses the existing method that queries `img[alt="delete"][name="DOMAIN_NAME"]` → value attribute.
8. Return the zone ID string and nil error.

Error wrapping pattern: `fmt.Errorf("add zone %q: click trigger: %w", domainName, err)`

**DeleteZone(zoneID string, zoneName string) error**

Steps:
1. Navigate to zone list: call `zp.NavigateToZoneList()`. Return error if it fails.
2. Register dialog handler BEFORE the click (CRITICAL — dns.he.net calls prompt() which must be handled pre-emptively):
   ```go
   zp.page.On("dialog", func(dialog playwright.Dialog) {
       _ = dialog.Fill("DELETE")
       _ = dialog.Accept()
   })
   ```
   Note: playwright-go uses `page.On("dialog", handler)` not `page.OnDialog`. The handler func receives `playwright.Dialog`.
3. Build the selector for the delete image of this specific zone: `fmt.Sprintf(`img[alt="delete"][value="%s"]`, zoneID)`.
4. Click that image: `zp.page.Locator(selector).Click()`. Return wrapped error.
5. Wait for networkidle.
6. Verify zone is gone: attempt `zp.GetZoneID(zoneName)`. If it returns an error (zone not found), the deletion succeeded — return nil. If it returns a non-empty ID, return `fmt.Errorf("delete zone %q: zone still present after deletion", zoneName)`.

**GetZoneName(zoneID string) (string, error)**

Steps:
1. Build selector: `fmt.Sprintf(`img[alt="delete"][value="%s"]`, zoneID)`.
2. Get name attribute: `zp.page.Locator(selector).GetAttribute("name")`.
3. Return name and any error, wrapped: `fmt.Errorf("get zone name for ID %q: %w", zoneID, err)`.

**ListZones handler** — `func ListZones(db *sql.DB, sm *browser.SessionManager) http.HandlerFunc`

Implementation (follow accounts.go pattern exactly):
1. `claims := middleware.ClaimsFromContext(r.Context())` — if nil, return 401.
2. Log start: `slog.InfoContext(r.Context(), "list zones", "account_id", claims.AccountID)`.
3. Record `start := time.Now()`.
4. `var zones []model.Zone` — declared outside closure.
5. `err := sm.WithAccount(r.Context(), claims.AccountID, func(page playwright.Page) error { ... })`:
   - Inside closure: `zl := pages.NewZoneListPage(page)`.
   - `if err := zl.NavigateToZoneList(); err != nil { return err }`.
   - `list, err := zl.ListZones(); if err != nil { return err }`.
   - Populate AccountID on each zone: `for i := range list { list[i].AccountID = claims.AccountID }`.
   - Assign: `zones = list; return nil`.
6. After WithAccount: check errors:
   - `errors.Is(err, browser.ErrQueueTimeout)` → 429 with code "queue_timeout".
   - `errors.Is(err, browser.ErrSessionUnhealthy)` → 503 with code "session_unhealthy".
   - Any other non-nil err → 500 with code "browser_error".
7. `slog.InfoContext(r.Context(), "list zones done", "account_id", claims.AccountID, "count", len(zones), "duration_ms", time.Since(start).Milliseconds())`.
8. Build response: convert `[]model.Zone` to `[]ZoneResponse`, set FetchedAt to start time.
9. `w.Header().Set("Content-Type", "application/json"); w.WriteHeader(http.StatusOK); json.NewEncoder(w).Encode(map[string]any{"zones": result})`.

**CreateZone handler** — `func CreateZone(db *sql.DB, sm *browser.SessionManager) http.HandlerFunc`

Request body: `{ "name": "example.com" }`.

Implementation:
1. Decode request body into struct `{ Name string \`json:"name"\` }`. Return 400 on decode error.
2. Validate: Name must be non-empty and <= 253 chars. Return 400 if invalid.
3. `claims := middleware.ClaimsFromContext(r.Context())`.
4. `start := time.Now()`.
5. `var result ZoneResponse` — declared outside closure.
6. `err := sm.WithAccount(...)`:
   - `zl := pages.NewZoneListPage(page)`.
   - IDEMPOTENCY PRE-CHECK: `if err := zl.NavigateToZoneList(); err != nil { return err }`.
   - `existingID, lookupErr := zl.GetZoneID(req.Name)`.
   - If lookupErr == nil AND existingID != "": zone already exists. Populate result with existing zone, set a sentinel to signal 200 instead of 201. Return nil (not an error).
   - Otherwise: call `zoneID, err := zl.AddZone(req.Name)`. Return err if non-nil.
   - Populate result: `ZoneResponse{ID: zoneID, Name: req.Name, AccountID: claims.AccountID, FetchedAt: time.Now()}`.
7. Error handling same as ListZones.
8. `slog.InfoContext(r.Context(), "create zone done", "account_id", claims.AccountID, "zone_id", result.ID, "duration_ms", time.Since(start).Milliseconds())`.
9. Response: if zone pre-existed, write 200; if newly created, write 201. Body: `json.NewEncoder(w).Encode(result)`.

To communicate "pre-existed" from the closure to the outer handler, use a boolean flag declared before the `sm.WithAccount` call: `var existed bool`. Inside the closure, set `existed = true` on the idempotency path.
