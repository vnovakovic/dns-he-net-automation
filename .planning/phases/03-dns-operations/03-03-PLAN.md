**records.go** — package `validate`

Imports: `errors`, `fmt`, `net`, `strings`, `github.com/vnovakovic/dns-he-net-automation/internal/model`

**Allowed TTL values** (the complete set dns.he.net permits in the TTL select):
```go
var allowedTTLs = map[int]bool{
    300: true, 900: true, 1800: true, 3600: true,
    7200: true, 14400: true, 28800: true, 43200: true,
    86400: true, 172800: true,
}
```

**v1 allowed types** (same set as in handlers/records.go — replicated here to keep validation package self-contained):
```go
var v1Types = map[model.RecordType]bool{
    model.RecordTypeA:     true,
    model.RecordTypeAAAA:  true,
    model.RecordTypeCNAME: true,
    model.RecordTypeMX:    true,
    model.RecordTypeTXT:   true,
    model.RecordTypeSRV:   true,
    model.RecordTypeCAA:   true,
    model.RecordTypeNS:    true,
}
```

**ValidateRecord(rec model.Record) error**

Validation steps in order:

1. **Type check**: if `!v1Types[rec.Type]` → return `fmt.Errorf("unsupported record type %q: must be one of A, AAAA, CNAME, MX, TXT, SRV, CAA, NS", rec.Type)`.

2. **Name check**: if `strings.TrimSpace(rec.Name) == ""` → return `errors.New("name is required")`.
   If `len(rec.Name) > 253` → return `errors.New("name must not exceed 253 characters")`.

3. **TTL check**: if `!allowedTTLs[rec.TTL]` → return `fmt.Errorf("invalid TTL %d: must be one of 300, 900, 1800, 3600, 7200, 14400, 28800, 43200, 86400, 172800", rec.TTL)`.

4. **Type-specific checks**:

   **A**:
   - `ip := net.ParseIP(rec.Content)` — if nil or ip.To4() == nil → return `fmt.Errorf("invalid IPv4 address %q", rec.Content)`.

   **AAAA**:
   - `ip := net.ParseIP(rec.Content)` — if nil → return `fmt.Errorf("invalid IPv6 address %q", rec.Content)`.
   - If `ip.To4() != nil` → return `fmt.Errorf("%q is an IPv4 address, not IPv6", rec.Content)` (rejects mapping of IPv4 into AAAA).

   **CNAME, NS**:
   - `if strings.TrimSpace(rec.Content) == ""` → return `errors.New("content is required for CNAME/NS records")`.

   **MX**:
   - `if strings.TrimSpace(rec.Content) == ""` → return `errors.New("MX content (mail server hostname) is required")`.
   - `if rec.Priority < 1 || rec.Priority > 65535` → return `fmt.Errorf("MX priority must be 1-65535, got %d", rec.Priority)`.

   **TXT**:
   - `if strings.TrimSpace(rec.Content) == ""` → return `errors.New("TXT content is required")`.
   - (No length limit enforced here — dns.he.net handles its own TXT length constraints.)

   **SRV**:
   - `if rec.Priority < 1 || rec.Priority > 65535` → return `fmt.Errorf("SRV priority must be 1-65535, got %d", rec.Priority)`.
   - `if rec.Weight < 0 || rec.Weight > 65535` → return `fmt.Errorf("SRV weight must be 0-65535, got %d", rec.Weight)`.
   - `if rec.Port < 1 || rec.Port > 65535` → return `fmt.Errorf("SRV port must be 1-65535, got %d", rec.Port)`.
   - `if strings.TrimSpace(rec.Target) == ""` → return `errors.New("SRV target is required")`.

   **CAA**:
   - `if strings.TrimSpace(rec.Content) == ""` → return `errors.New("CAA content is required")`.
   - Basic format check — CAA content must have at least 3 space-separated tokens (flags tag value):
     `parts := strings.Fields(strings.TrimSpace(rec.Content)); if len(parts) < 3 { return errors.New("CAA content must be in format: flags tag value") }`.

5. Return nil on success.

// WriteJSON writes a JSON success response with the given HTTP status and body value.

func WriteJSON(w http.ResponseWriter, status int, v any) {
    w.Header().Set("Content-Type", "application/json")

if filterType := r.URL.Query().Get("type"); filterType != "" {
    ft := model.RecordType(strings.ToUpper(strings.TrimSpace(filterType)))
    filtered := records[:0]
    for _, rec := range records {
        if rec.Type == ft {
            filtered = append(filtered, rec)
        }
    }
    records = filtered

if filterName := r.URL.Query().Get("name"); filterName != "" {
    fn := strings.ToLower(strings.TrimSpace(filterName))
    filtered := records[:0]
    for _, rec := range records {

            filtered = append(filtered, rec)
        }
    }
    records = filtered
}
