**ParseRecordRow(rowID string) (*model.Record, error)**

Method signature: `func (zp *ZoneListPage) ParseRecordRow(rowID string) (*model.Record, error)`

Steps:
1. Build row selector: `fmt.Sprintf("tr#%s", rowID)`.
2. Get all tds from this row: `cells := zp.page.Locator(fmt.Sprintf("tr#%s td", rowID))`.
3. Count must be 10; if not return `fmt.Errorf("parse record row %q: expected 10 tds, got %d", rowID, count)`.
4. Extract each cell's InnerText:
   - `recordID` = td[1].InnerText()  (strip whitespace with strings.TrimSpace)
   - `name`     = td[2].InnerText()
   - `recType`  = td[3].InnerText()
   - `ttlStr`   = td[4].InnerText()
   - `prioStr`  = td[5].InnerText()
   - `content`  = td[6].InnerText()
   - `ddnsStr`  = td[7].InnerText()
5. Parse TTL: `ttl, err := strconv.Atoi(strings.TrimSpace(ttlStr))` — return error on parse failure.
6. Parse priority: if `strings.TrimSpace(prioStr) == "-"` then priority = 0; otherwise `strconv.Atoi`.
7. Parse DDNS: `dynamic := strings.TrimSpace(ddnsStr) == "1"`.
8. Build record:
   ```go
   rec := &model.Record{
       ID:       strings.TrimSpace(recordID),
       Type:     model.RecordType(strings.TrimSpace(recType)),
       Name:     strings.TrimSpace(name),
       TTL:      ttl,
       Priority: priority,
       Dynamic:  dynamic,
   }
   ```
9. For SRV records: content field holds "Weight Port Target" space-separated. Split by space:
   ```go
   if rec.Type == model.RecordTypeSRV {
       parts := strings.Fields(strings.TrimSpace(content))
       if len(parts) == 3 {
           rec.Weight, _ = strconv.Atoi(parts[0])
           rec.Port, _   = strconv.Atoi(parts[1])
           rec.Target    = parts[2]
           rec.Content   = "" // SRV has no Content field
       } else {
           // Fallback: store raw content for debugging
           rec.Content = strings.TrimSpace(content)
       }
   } else {
       rec.Content = strings.TrimSpace(content)
   }
   ```
10. Return rec, nil.

**FindRecord(zoneID string, rec model.Record) (string, error)**

Method signature: `func (zp *ZoneListPage) FindRecord(zoneID string, rec model.Record) (string, error)`

Returns the record ID of the first matching record, or "" if not found.
Used for idempotency: before creating, check if an identical record already exists.

Steps:
1. `all, err := zp.ListRecords(zoneID); if err != nil { return "", err }`.
2. Match criteria by type:
   - **All types**: Type must match AND Name must match (case-insensitive comparison: `strings.EqualFold`).
   - **A, AAAA, CNAME, NS, TXT, CAA**: also match Content (strings.EqualFold for domain names, exact for IP/TXT).
   - **MX**: match Content AND Priority.
   - **SRV**: match Priority, Weight, Port, and Target (strings.EqualFold for Target).
3. Return the ID of the first match, or "" with nil error if no match found.
4. Error is only returned for scraping failures, not for "not found".

Matching logic — use a helper `recordsMatch(existing, target model.Record) bool`:
```go
func recordsMatch(a, b model.Record) bool {
    if a.Type != b.Type { return false }
    if !strings.EqualFold(a.Name, b.Name) { return false }
    switch a.Type {
    case model.RecordTypeMX:
        return strings.EqualFold(a.Content, b.Content) && a.Priority == b.Priority
    case model.RecordTypeSRV:
        return a.Priority == b.Priority && a.Weight == b.Weight &&
               a.Port == b.Port && strings.EqualFold(a.Target, b.Target)
    default:
        return strings.EqualFold(a.Content, b.Content)
    }
}
```

    switch {
    case errors.Is(err, browser.ErrQueueTimeout):
        response.WriteError(w, http.StatusTooManyRequests, "queue_timeout", "Request queue full, retry later")
    case errors.Is(err, browser.ErrSessionUnhealthy):
        response.WriteError(w, http.StatusServiceUnavailable, "session_unhealthy", "Browser session unavailable")
    default:
        response.WriteError(w, http.StatusInternalServerError, "browser_error", "Browser operation failed")

}
