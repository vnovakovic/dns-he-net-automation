**Step 1 — Create migration file:**

Create `internal/store/migrations/003_audit_log.sql` following the exact goose format used in 001_init.sql and 002_tokens.sql (read those files first to confirm format):

```sql
-- +goose Up
CREATE TABLE IF NOT EXISTS audit_log (
    id           INTEGER  PRIMARY KEY AUTOINCREMENT,
    created_at   DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    token_id     TEXT     NOT NULL,
    account_id   TEXT     NOT NULL,
    action       TEXT     NOT NULL CHECK (action IN ('create','update','delete','sync')),
    resource     TEXT     NOT NULL,
    result       TEXT     NOT NULL CHECK (result IN ('success','failure')),
    error_msg    TEXT
);

CREATE INDEX IF NOT EXISTS idx_audit_log_account_id ON audit_log(account_id);
CREATE INDEX IF NOT EXISTS idx_audit_log_created_at  ON audit_log(created_at);
CREATE INDEX IF NOT EXISTS idx_audit_log_token_id    ON audit_log(token_id);

-- +goose Down
DROP INDEX IF EXISTS idx_audit_log_token_id;
DROP INDEX IF EXISTS idx_audit_log_created_at;
DROP INDEX IF EXISTS idx_audit_log_account_id;
DROP TABLE IF EXISTS audit_log;
```

```go
type Entry struct {
    TokenID   string
    AccountID string
    Action    string  // "create" | "update" | "delete" | "sync"
    Resource  string  // "record:<id>" | "zone:<id>"
```

- Return the error from db.ExecContext (never silently discard)

**Pattern for every mutating handler:**
After the browser operation returns (whether success or error), build the audit entry and call Write. NEVER short-circuit on audit failure — only log it:

```go
auditResult := "success"
auditErrMsg := ""
if operationErr != nil {
    auditResult = "failure"
    auditErrMsg = operationErr.Error()
}
if err := audit.Write(r.Context(), db, audit.Entry{
    TokenID:   claims.ID,      // JWT token ID (jti claim)
    AccountID: claims.AccountID,
    Action:    "create",       // varies per handler
    Resource:  "zone:" + zoneID,  // or "record:" + recordID
    Result:    auditResult,
    ErrorMsg:  auditErrMsg,
}); err != nil {
    slog.ErrorContext(r.Context(), "audit log write failed", "error", err)
    // DO NOT return HTTP error — audit failure is non-fatal
}
```

**zones.go — add audit calls to:**
- `CreateZone` handler: Action "create", Resource "zone:" + zoneID (the zone ID returned by the browser op)
- `DeleteZone` handler: Action "delete", Resource "zone:" + chi.URLParam(r, "zoneID")
