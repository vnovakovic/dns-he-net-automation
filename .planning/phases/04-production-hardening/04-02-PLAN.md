**internal/resilience/retry.go**

Package `resilience`. Exports `WithRetry(ctx, op) error`.

```go
// WithRetry wraps a browser operation with exponential backoff and jitter.
// Only transient errors are retried (timeout, session unhealthy, deadline exceeded).
// Permanent errors (e.g., validation failures, zone-not-found) return immediately.
// Max 3 attempts: initial + 2 retries. Backoff: 500ms base, 200ms jitter.
func WithRetry(ctx context.Context, op func(context.Context) error) error
```

Implementation:
- `b := retry.NewExponential(500 * time.Millisecond)`
- `b = retry.WithJitter(200*time.Millisecond, b)`
- `b = retry.WithMaxRetries(3, b)`
- `retry.Do(ctx, b, func(ctx context.Context) error { ... })`
- Inside the Do callback: call op(ctx), if err != nil and isTransientBrowserError(err) return `retry.RetryableError(err)`, else return err directly

`isTransientBrowserError(err) bool`:
- `errors.Is(err, browser.ErrSessionUnhealthy)` — session recovery failure
- `errors.Is(err, context.DeadlineExceeded)` — operation timeout
- `strings.Contains(err.Error(), "timeout")` — playwright timeout strings
- `strings.Contains(err.Error(), "Target closed")` — Chromium crash
- Returns false for all other errors (logic errors, validation errors, not-found)

Do NOT retry Vault credential errors — those are handled separately by the stale cache (research Pitfall 5).

**internal/api/middleware/ratelimit.go**

Package `middleware`. Exports `PerTokenRateLimit(requestsPerMin int)` and `GlobalRateLimit(requestsPerMin int)`.

```go
// PerTokenRateLimit returns chi middleware that limits requests per bearer token.
// Must be registered AFTER BearerAuth middleware (needs token in Authorization header).
// Returns 429 with Retry-After header and JSON error body on limit exceeded.
func PerTokenRateLimit(requestsPerMin int) func(http.Handler) http.Handler {
    return httprate.Limit(
        requestsPerMin,
        time.Minute,
        httprate.WithKeyFuncs(func(r *http.Request) (string, error) {
            header := r.Header.Get("Authorization")
            if strings.HasPrefix(header, "Bearer ") {
                return strings.TrimPrefix(header, "Bearer "), nil
            }
            return r.RemoteAddr, nil
        }),
        httprate.WithLimitHandler(func(w http.ResponseWriter, r *http.Request) {
            response.WriteError(w, http.StatusTooManyRequests, "rate_limited",
                "Request rate limit exceeded. Check the Retry-After header.")
        }),
    )
}

// GlobalRateLimit returns chi middleware limiting total requests across all clients.
// Register BEFORE BearerAuth (provides DDoS protection before auth overhead).
func GlobalRateLimit(requestsPerMin int) func(http.Handler) http.Handler {
    return httprate.LimitAll(requestsPerMin, time.Minute)
}
```

Import `response "github.com/vnovakov/dns-he-net-automation/internal/api/response"` for WriteError.

```go
// OLD:
if elapsed < sm.minOpDelay {
    select {
    case <-time.After(sm.minOpDelay - elapsed):
    ...

// NEW (BROWSER-08 jitter):
if elapsed < sm.maxOpDelay {
    // Jitter: random delay between minOpDelay and maxOpDelay.
    // math/rand (not crypto/rand) is sufficient — this is rate-limiting, not security.
    jitterRange := sm.maxOpDelay - sm.minOpDelay
    jitter := time.Duration(rand.Int63n(int64(jitterRange + 1)))
    target := sm.minOpDelay + jitter
    remaining := target - elapsed
    if remaining > 0 {
        select {
        case <-time.After(remaining):
        case <-opCtx.Done():
            return opCtx.Err()
        }
    }
}
```
